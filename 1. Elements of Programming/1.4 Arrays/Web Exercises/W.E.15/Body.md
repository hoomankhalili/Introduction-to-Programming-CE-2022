بنر. یک برنامه به نام Banner.java بنویسید که یک رشته را به عنوان آرگومان خط فرمان بگیرد و رشته را با حروف بزرگ مانند زیر چاپ کند.
````
% java Banner "Kevin"
 #    #  ######  #    #     #    #    #
 #   #   #       #    #     #    ##   #
 ####    #####   #    #     #    # #  #
 #  #    #       #    #     #    #  # #
 #   #   #        #  #      #    #   ##
 #    #  ######    ##       #    #    #
 ````
 بنر ابزار Unix را تقلید می کند.
